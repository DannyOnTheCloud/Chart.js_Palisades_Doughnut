Palisades Fire Structure Analysis

This analysis was created using Chart.js, a flexible and open-source JavaScript library for creating interactive data visualizations. The data was sourced from the California Open Data Portal: DINS 2025 Palisades Public View. Updated on January 24, 2025, at 4:00 PM (UTC-08:00).

The chart below illustrates the distribution of structure types affected by the Palisades Fire, which began on Tuesday, January 7, 2025, at approximately 10:30 AM PST. This visualization provides critical insights into the types of infrastructure impacted, helping guide future prevention and recovery efforts.

The color rectangles are interactive; click on them to eliminate them from the chart and see how it adjusts.
